# CounterStrike
## Setup for Starting System (Laser+LIDAR)
1. Connect the 2 USB cables.
2. Open Arduino IDE.
3. Select Board > COM5 > Nano Arduino > OK.
4. File > Examples > Firmata > StandardFirmata.
5. Tools > Processor: "..." > "..." (Old Bootloader) > Upload.
6. Run `system/main.py`.
